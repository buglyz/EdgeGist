## 🌟 核心功能

### 1. 混合存储策略
```typescript
// 小文件 → D1（快速）
if (fileSize < 100KB && isText) {
  store in D1
}

// 大文件/二进制 → R2（经济）
else {
  store in R2
}
```

### 2. 智能字数统计
```typescript
countWords("深度学习 deep learning 神经网络")
// → 12 中文字 + 2 英文词 = 14
```

### 3. Markdown + LaTeX 预览
```markdown
行内公式：$E = mc^2$
块级公式：$$\int_0^\infty e^{-x} dx$$
```

### 4. 标签管理
```bash
PUT /gists/{id}/tags
{"tags": ["论文", "机器学习", "CVPR2024"]}
```

### 5. DOCX 预览
- 服务端转换为 HTML
- XSS 防护
- 友好错误处理

## 🚀 部署清单

### 1. 本地开发 ✅
```bash
git clone https://github.com/buglyz/EdgeGist.git
cd EdgeGist
bun install
bun run dev
```

### 2. 数据库迁移 ✅
```bash
wrangler d1 migrations apply edge-gist --remote
```
**注意**: migration 0003 已修复 SQLite 语法错误

### 3. R2 存储桶 ✅
```bash
wrangler r2 bucket create edgegist-files
```

### 4. 配置文件 ✅
```bash
cp wrangler.example.jsonc wrangler.jsonc
# 编辑配置...
```

### 5. 部署 ✅
```bash
bun run build
wrangler deploy
```
